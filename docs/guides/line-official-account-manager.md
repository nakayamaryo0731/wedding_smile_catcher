# LINE Official Account Manager 利用手順

ブラウザからLINEメッセージを送信するための手順書です。

## 1. ログイン

1. <https://manager.line.biz/> にアクセス
2. 「ログイン」をクリック
3. LINE Developersで使用しているアカウントでログイン
   - LINEアカウント or ビジネスアカウント

## 2. アカウント選択

ログイン後、管理しているLINE公式アカウントの一覧が表示されます。

1. 「笑顔写真コンテスト」（または設定したボット名）を選択

## 3. メッセージ配信

### 一斉配信（ブロードキャスト）

1. 左メニューから **「メッセージ配信」** をクリック
2. **「メッセージを作成」** をクリック
3. メッセージを作成：
   - 配信先: 「すべての友だち」
   - 配信日時: 「今すぐ配信」または日時指定
   - メッセージ入力欄にテキストを入力

### メッセージ例（バイラル用）

```
🎉 先日はおめでとうございました！

「笑顔写真コンテスト」にご参加いただき
ありがとうございました 📸

━━━━━━━━━━━━━━━━━

このアプリはご自身の結婚式でもお使いいただけます！
簡単に使えるので、ぜひ結婚式を盛り上げましょう！

ご興味があればお気軽にお問い合わせください 👇
nakayamaryo0731@gmail.com
```

1. **「配信」** ボタンをクリック
2. 確認ダイアログで **「配信」** をクリック

## 4. 配信結果の確認

1. 左メニューから **「分析」** > **「メッセージ配信」**
2. 配信したメッセージの開封率・クリック率を確認

## 5. その他の機能

| 機能 | 説明 |
|------|------|
| リッチメッセージ | 画像付きのタップ可能なメッセージ |
| リッチメニュー | トーク画面下部の固定メニュー |
| クーポン | クーポン配信 |
| ショップカード | ポイントカード機能 |
| 応答メッセージ | キーワードに自動返信 |
| あいさつメッセージ | 友だち追加時の自動メッセージ |

## 注意事項

- 無料プランには月間メッセージ数の上限あり
- 一度送信したメッセージは取り消し不可
- テスト送信機能で事前確認を推奨

## トラブルシューティング

### ログインできない

- LINE Developersと同じアカウントか確認
- ブラウザのCookieをクリアして再試行

### アカウントが表示されない

- LINE Developersコンソールで、Messaging APIチャネルが正しく設定されているか確認
- 権限があるか確認（Admin権限が必要）
