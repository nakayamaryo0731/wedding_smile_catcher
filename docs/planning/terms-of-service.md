## 1. 利用規約

### 第1条（適用）

本規約は、本サービス「Wedding Smile Catcher」（以下「本サービス」）の利用に関する条件を定めるものです。利用者は、本規約に同意の上、本サービスを利用するものとします。

### 第2条（定義）

- 「顧客」: 本サービスにアカウントを作成し、イベントを主催する方
- 「ゲスト」: 顧客が開催するイベントに参加し、LINE Botを通じて写真を投稿する方
- 「イベント」: 顧客が本サービス上で作成する結婚式等のイベント

### 第3条（サービス内容）

本サービスは以下の機能を提供します:

1. LINE Botを通じた写真収集
2. AIによる笑顔スコアリング
3. リアルタイムランキング表示
4. 管理画面によるイベント管理

### 第4条（料金・支払い）

- イベント利用料: 15,000円（税込）/ 1イベント
- 支払い方法: 銀行振込 or ココナラ決済
- 支払いタイミング: イベント有効化前（支払い確認後に運営者がイベントを有効化）

### 第5条（サポート）

- **イベント当日のリアルタイムサポートは提供しません**
- 問い合わせ対応: メールベース、平日対応
- 事前テスト: 顧客はイベント1週間前までに動作確認テストを完了してください

### 第6条（返金ポリシー）

| ケース | 対応 |
|--------|------|
| 当社に起因するシステム障害によりサービスが利用できなかった場合 | 全額返金 |
| 外部サービス（GCP、LINE等）の障害に起因する場合 | **要検討**（半額返金 or 免責）。※フォールバック設計により完全停止の可能性は低い |
| 顧客の設定ミス・操作ミスに起因する場合 | 返金なし |
| イベント7日前までのキャンセル | 全額返金 |
| イベント7〜3日前のキャンセル | 半額返金 |
| イベント3日前以降のキャンセル | 返金なし |

### 第7条（禁止事項）

以下の行為を禁止します:

1. 違法な目的での利用
2. 他者のプライバシーを侵害する行為
3. 不正アクセス、サービスへの攻撃
4. 本サービスのリバースエンジニアリング
5. 公序良俗に反する画像の投稿

### 第8条（免責事項）

1. 本サービスは「現状有姿（as-is）」で提供されます
2. 外部サービス（Google Cloud Platform、LINE Messaging API等）の障害、仕様変更等に起因する問題について、当社は責任を負いません
3. AIによるスコアリング結果の正確性は保証しません
4. ネットワーク障害、端末の不具合等に起因する問題は免責とします

### 第9条（データの取り扱い）

- 詳細はプライバシーポリシーに定めます
- イベント終了後30日経過後、画像データを自動削除します
- 顧客はイベント終了後30日以内にデータをダウンロードしてください

### 第10条（サービスの変更・終了）

- 本サービスの内容を予告なく変更する場合があります
- サービス終了の場合は30日前までに通知します

### 第11条（規約の変更）

- 本規約は随時変更される場合があります
- 重要な変更の場合はメールで通知します

### 第12条（未成年の利用）

- 未成年の顧客（アカウント作成者）は、保護者の同意を得た上で本サービスを利用するものとします
- ゲストとしての参加（LINE Botへの写真投稿）については、顧客がゲストへの説明・同意取得の責任を負います
- 未成年のゲストについては、顧客が当該ゲストの保護者に対して本サービスの内容（AI による顔写真の処理を含む）を説明し、了承を得るものとします

### 第13条（準拠法・管轄）

- 準拠法: 日本法
- 管轄裁判所: [運営者住所地の裁判所]

## 2. プライバシーポリシー

### 収集する個人情報

| 対象 | 情報 | 利用目的 |
|------|------|---------|
| 顧客 | メールアドレス | アカウント管理、連絡 |
| 顧客 | 表示名 | サービス内表示 |
| ゲスト | 名前（フルネーム） | ランキング表示 |
| ゲスト | 顔写真 | AIスコアリング、ランキング表示 |
| ゲスト | LINE User ID | LINE Bot連携 |

### 利用目的

収集した個人情報は以下の目的にのみ利用します:

1. 本サービスの提供・運営
2. AIによる画像スコアリング
3. ランキング画面への表示
4. 顧客への連絡・通知
5. サービスの改善・分析（統計的処理のみ）

### 第三者提供

個人情報を第三者に提供することはありません。ただし以下は除きます:

- Google Cloud Platform（画像処理・データ保存の委託先として）
- LINE（Bot通信の委託先として）
- 法令に基づく開示要請があった場合

### AIによる画像処理

- 投稿された画像はGoogle Cloud Vision APIおよびVertex AI（Gemini）で処理されます
- 処理目的: 顔検出（笑顔の度合いの判定）、テーマ評価（結婚式との関連性評価）
- Google は画像データを一時的にメモリ上で処理し、保存しません
- Google は顧客データをモデル学習に使用しません
- Google のデータ処理ポリシーに従います

### メッセージの取消（unsend）

- ゲストがLINE上で写真メッセージを取り消した場合、当該画像をCloud Storageおよび関連するスコアデータから削除します

### サービス終了時のデータ処理

- 本サービスを終了する場合、全ユーザーデータ（画像、スコア、アカウント情報）を速やかに削除します
- サービス終了の通知後、30日間のデータダウンロード期間を設けます

### データの保存期間

| データ | 保存期間 |
|--------|---------|
| 画像データ | イベント終了後30日間 |
| スコア等メタデータ | イベント終了後30日間 |
| 顧客アカウント情報 | アカウント削除まで |

### データの削除

- 顧客はいつでもアカウントとデータの削除を要求できます
- 削除要求は30日以内に処理します
- 問い合わせ先: [メールアドレス]

### Cookie

- Firebase SDKによる技術的Cookieのみ使用
- トラッキング目的のCookieは使用しません
