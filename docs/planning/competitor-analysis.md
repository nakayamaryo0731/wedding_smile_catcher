# 競合分析

Wedding Smile Catcher の競合・類似サービスを分析し、差別化ポイントを明確にする。

最終更新: 2026-01-26

---

## 1. 競合カテゴリ

### A. 結婚式向け写真共有サービス

| サービス | 概要 | 価格帯 |
|---------|------|--------|
| Foto | 結婚式写真共有アプリ | 無料〜 |
| 30days Album（デイズ） | 写真共有・ダウンロード | 無料〜 |
| 結婚式用LINEアルバム | LINEの既存機能を活用 | 無料 |

### B. イベント向けフォトサービス

| サービス | 概要 | 価格帯 |
|---------|------|--------|
| Snapchat / Instagram フィルター | カスタムフィルターで写真撮影 | 数千円〜 |
| フォトブース | 写真撮影スペース + 印刷 | 数万円〜 |

### C. 結婚式余興サービス

| サービス | 概要 | 価格帯 |
|---------|------|--------|
| MarryGiftMovie | ムービー系余興 | 数万円〜 |
| 各種クイズ / ゲームアプリ | ゲスト参加型ゲーム | 無料〜数千円 |

## 2. 差別化ポイント

### Wedding Smile Catcher の独自性

| ポイント | 本サービス | 一般的な写真共有サービス |
|---------|-----------|---------------------|
| **ゲーミフィケーション** | AIがスコアをつけて競争要素がある | 写真を集めるだけ |
| **交流促進** | 大人数ほど高スコア → 自然にグループが生まれる | なし |
| **LINEベース** | 新しいアプリ不要、誰でも使える | 専用アプリが必要な場合も |
| **リアルタイム性** | スクリーンでランキングが更新される | 後からアルバムを見る |
| **AI活用** | 笑顔検出 + テーマ評価 | なし |
| **余興としての価値** | 「一斉投稿タイム」で盛り上がる演出 | 余興にはならない |

### 直接競合が少ない理由

- 「結婚式 × AI笑顔スコアリング × LINE Bot」の組み合わせは独自性が高い
- 類似の個人開発サービスはほぼ見当たらない（元となったシステムは公開されていない）
- 大手がこのニッチ市場に参入する可能性は低い

---

## 3. 弱み・リスク

| 弱み | 対策 |
|------|------|
| 知名度ゼロ | Phase 1で実績作り → ココナラでレビュー獲得 |
| 当日サポートなし | 事前テスト必須化、分かりやすい利用ガイド |
| 個人開発のため信頼性への懸念 | 利用規約の整備、返金ポリシー明示 |
| 写真共有は無料の代替手段がある | 「余興」「ゲーミフィケーション」で差別化、写真共有とは違うカテゴリ |

## 4. ポジショニング

```
                高い

       価値      │

                 │

                低い
              シンプル ──────── インタラクティブ
```

**ポジション**: 「写真共有」ではなく「ゲスト参加型デジタル余興」

---

## 5. ココナラでの競合

### 「結婚式」で検索した場合の主な出品カテゴリ

- ムービー制作（プロフィールムービー、余興ムービー）: 多数
- 招待状デザイン: 多数
- ウェルカムボード制作: 多数
- デジタル余興・ゲーム: **ほぼなし**

→ **ブルーオーシャンの可能性**。ただし需要があるかの検証が必要。
