## 2. ターゲットと獲得チャネル

### ターゲット
- 結婚式を3〜6ヶ月以内に控えるカップル
- 「余興どうしよう」「写真を集めたい」という課題を持つ人

### チャネル優先度

| 優先度 | チャネル | 理由 | コスト |
|--------|---------|------|--------|
| ★★★ | ココナラ | 能動的に探している人にリーチ。競合ほぼなし | 手数料22% |
| ★★★ | SEO（LP最適化）| 検索経由で継続的にリーチ | 無料 |
| ★★☆ | Instagram | 結婚式準備中のカップルが情報収集で活用 | 無料 |
| ★★☆ | X | 拡散力。エンジニア層への認知にも有効 | 無料 |
| ★☆☆ | Google広告 | 「結婚式 余興」等のキーワード。少額テスト | 5,000〜10,000円/月 |
| ★☆☆ | ウェディング系メディア | 掲載されれば効果大だが、時間がかかる | 無料〜 |

## 4. SNS投稿内容案

個人アカウントなので、週1〜2回、自然な形で。無理に毎日投稿しない。

### X（メイン）

フォロワー600人。エンジニア層への拡散力がある。

**投稿テーマ例:**
1. サービスローンチ報告（「個人開発で結婚式向けサービス作った」系は伸びやすい）
2. 開発の裏話（AI × LINE Bot × 結婚式）
3. 実績の共有（368枚、参加率60%）
4. 技術スタック紹介

### Instagram（サブ）

結婚式準備中のカップルがハッシュタグ検索で見つける可能性。

**ハッシュタグ例:**
`#結婚式 #結婚式準備 #プレ花嫁 #余興 #結婚式余興 #ウェディング #披露宴 #結婚式アイデア`

**投稿テーマ例:**
1. サービス紹介（ランキング画面スクショ + 概要）
2. 使い方3ステップ（カルーセル投稿）
3. 実績紹介

## 5. SEO対策キーワード

| キーワード | 検索意図 | 難易度（推定）|
|-----------|---------|-------------|
| 結婚式 余興 | 余興のアイデアを探している | 中 |
| 結婚式 写真 ゲーム | 写真系の余興を探している | 低 |
| 結婚式 盛り上がる 演出 | 演出アイデアを探している | 中 |
| 結婚式 LINE ゲーム | LINE活用の余興を探している | 低 |
| ゲスト参加型 余興 | 全員参加の余興を探している | 低 |
| 結婚式 AI | AI活用の結婚式サービスを探している | 低 |

→ C-2 のメタタグ最適化でこれらを反映する。
