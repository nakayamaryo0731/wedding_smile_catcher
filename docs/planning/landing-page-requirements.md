# ランディングページ / 販売ページ要件定義

本サービスを顧客に訴求するためのページ構成と内容を定義する。
Phase 2（ココナラ出品）の商品説明ページと、Phase 3（自前LP）の両方に使える内容とする。

最終更新: 2026-01-26

---

## 1. ターゲット

- 結婚式を控えるカップル
- 「結婚式を盛り上げたい」「写真をたくさん集めたい」という課題を持つ人
- IT リテラシーは問わない（セルフサービスで使える設計）

---

## 2. 訴求ポイント（刺さるメッセージ）

### メインキャッチコピー案

- 「結婚式の笑顔、AIがスコアにします」
- 「ゲスト全員が写真を撮りたくなる結婚式余興」
- 「LINEで送るだけ。AIが笑顔を採点する新しい余興」

### サブコピー

- 「ゲストの60%が参加した実績あり」
- 「準備はQRコードを配るだけ」
- 「スクリーンにリアルタイムでランキング表示」

## 3. ページ構成
次に進もう
### セクション1: ヒーロー

- キャッチコピー
- 式場スクリーンにランキングが映っているイメージ画像
- CTA（購入 / 問い合わせ）

### セクション2: こんなお悩みありませんか？

- 結婚式で撮った写真が新郎新婦に届かない
- ゲスト同士の交流がなかなか生まれない
- 余興のネタに困っている
- 写真共有サービスは使いにくい（特に年配の方）

### セクション3: サービスの特徴

| 特徴 | 説明 |
|------|------|
| LINEだから誰でも使える | アプリ不要、QRコードを読むだけ |
| AIが笑顔を自動採点 | 笑顔の度合い×人数×テーマ評価 |
| リアルタイムランキング | スクリーンに表示、盛り上がる |
| 大人数ほど高スコア | グループ間の交流が自然に生まれる |

### セクション4: 使い方（3ステップ）

```
Step 1: イベントを作成（5分）
  └── 日付と名前を入力するだけ

Step 2: 受付でQRコードを配布
  └── 自動生成されるQRコードを印刷して配るだけ

Step 3: 当日はスクリーンに表示するだけ
  └── あとは自動で盛り上がります
```

### セクション5: 実績・数字

- 元となったシステムでの実績:
  - ゲスト参加率: **60%**
  - 収集写真数: **368枚**
  - システム稼働: **ゼロダウンタイム**

### セクション6: 料金

- **15,000円（税込）/ 1イベント**
- 含まれるもの:
  - LINE Bot利用
  - AIスコアリング（無制限）
  - ランキング画面
  - 管理画面
  - 画像保存（30日間）
  - データダウンロード

### セクション7: よくある質問

| 質問 | 回答 |
|------|------|
| ゲストはアプリをインストールする必要がありますか？ | いいえ、LINEだけで利用できます |
| 何人まで参加できますか？ | 制限はありません |
| 当日のサポートはありますか？ | 当日のリアルタイムサポートは提供していません。事前テストで動作確認をお願いしています |
| 写真のデータはもらえますか？ | はい、管理画面からダウンロードできます（イベント後30日以内） |
| スクリーンがなくても使えますか？ | はい、LINE Botでの写真投稿・スコア返信は使えます。ランキング表示にはスクリーンまたはPC/タブレットが必要です |
| キャンセルはできますか？ | イベント7日前まで全額返金、7〜3日前は半額返金、3日前以降は返金不可 |

### セクション8: CTA

- 購入ボタン / 問い合わせボタン
- 「まずはアカウント作成（無料）」

- 最初は **9,800円** で実績・レビューを貯める
