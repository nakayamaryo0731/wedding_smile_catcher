## 3. ページ構成

### セクション1: ヒーロー

- キャッチコピー
- 式場スクリーンにランキングが映っているイメージ画像
- CTA（購入 / 問い合わせ）

### セクション2: こんなお悩みありませんか？

- 結婚式で撮った写真が新郎新婦に届かない
- ゲスト同士の交流がなかなか生まれない
- 余興のネタに困っている
- 写真共有サービスは使いにくい（特に年配の方）

### セクション3: サービスの特徴

| 特徴 | 説明 |
|------|------|
| LINEだから誰でも使える | アプリ不要、QRコードを読むだけ |
| AIが笑顔を自動採点 | 笑顔の度合い×人数×テーマ評価 |
| リアルタイムランキング | スクリーンに表示、盛り上がる |
| 大人数ほど高スコア | グループ間の交流が自然に生まれる |

### セクション4: 使い方（3ステップ）

```
Step 1: イベントを作成（5分）
  └── 日付と名前を入力するだけ

Step 2: 受付でQRコードを配布
  └── 自動生成されるQRコードを印刷して配るだけ

Step 3: 当日はスクリーンに表示するだけ
  └── あとは自動で盛り上がります
```

### セクション5: デモ / 実績

**Phase 1:**
- デモ動画またはスクリーンショットのギャラリー
- 実際の画面を見せてイメージを掴んでもらう

**Phase 2以降（実績が貯まってから）:**
- ゲスト参加率: **60%**
- 収集写真数: **368枚**
- システム稼働: **ゼロダウンタイム**

### セクション6: 料金

**Phase 1（初期）:**
- **無料** - 実績作りのため初期は無料提供
- 「**先着5組限定で無料**」

**Phase 2以降:**
- **15,000円（税込）/ 1イベント**

**含まれるもの:**
- LINE Bot利用
- AIスコアリング（無制限）
- ランキング画面
- 管理画面
- 画像保存（30日間）
- データダウンロード

### セクション7: よくある質問

| 質問 | 回答 |
|------|------|
| ゲストはアプリをインストールする必要がありますか？ | いいえ、LINEだけで利用できます |
| 何人まで参加できますか？ | 制限はありません |
| 当日のサポートはありますか？ | 当日のリアルタイムサポートは提供していません。事前テストで動作確認をお願いしています |
| 写真のデータはもらえますか？ | はい、管理画面からダウンロードできます（イベント後30日以内） |
| スクリーンがなくても使えますか？ | はい、LINE Botでの写真投稿・スコア返信は使えます。ランキング表示にはスクリーンまたはPC/タブレットが必要です |
| 本当に無料ですか？ | はい、先着5組限定で完全無料です。サービス改善のため、ご利用後にアンケートへのご協力をお願いしています |

### セクション8: CTA

**Phase 1（初期）:**
- 「お問い合わせ」ボタン → Google Forms → スプレッドシートに記録 → email でやりとり

**Google Forms 項目:**
1. お名前（必須）
2. メールアドレス（必須）
3. 結婚式の予定日（必須）
4. ゲスト数（任意）- 選択式: 〜50人 / 50〜100人 / 100人〜
5. 式場名（任意）
6. 知ったきっかけ（任意）- 選択式: SNS / 検索 / 紹介 / その他
7. ご質問・ご要望（任意）- 自由記述

**Phase 2（ココナラ出品後）:**
- 「ココナラで購入」ボタン → ココナラ商品ページへ誘導
