## 1. ニッチSaaS（一回きり利用型）の初期顧客獲得

### このプロダクトの特殊性

| 項目 | 通常のSaaS | Wedding Smile Catcher |
|------|-----------|----------------------|
| 収益モデル | 月額/年額サブスク | 都度課金（1回きり） |
| LTV | 月額 × 継続月数 | 1回分の利用料のみ |
| 重要指標 | チャーン率、NRR | 新規獲得数/月、CAC |
| 成長ドライバー | リテンション | 常時新規獲得 |

最も近い業態は「フォトブースレンタル」「ケータリング」「イベント撮影」等のトランザクション型ビジネス。

### チャネル別評価

#### A. プランナー紹介ネットワーク
- **効果**: 高 / **コスト**: 成果報酬のみ / **時間**: 1〜3ヶ月
- ウェディング業界では紹介料として契約額の5〜20%が相場
- フリープランナーは差別化の演出を常に探している
- 紹介手数料（10〜15%）を設定し、プランナー側のリスクゼロにする

#### B. バイラルループ（参加ゲスト → 次の顧客）
- **効果**: 中〜高 / **コスト**: ゼロ / **時間**: 利用開始後すぐ蓄積
- 結婚式1回あたり50〜100人が体験（Zoomのバイラルループと同構造）
- リファラル経由のイベント参加者はコンバージョン率が2.5倍高い
- 両面報酬（紹介者にも被紹介者にも特典）が片面の2〜3倍効果的

**試算**: 月4件の利用 → 月2〜5件のリードが自動発生 → 転換率20%で月0.4〜1件の新規獲得（CAC = 0）

#### C. TikTok / Instagram
- **効果**: 中 / **コスト**: 時間のみ / **時間**: 1〜3ヶ月
- ある企業ではクライアントの60%がTikTok経由
- TikTokはフォロワー数に依存せず新規アカウントでもリーチしやすい
- Instagramのエンゲージメント率は低下傾向（2024年Q1の16.9% → 2025年Q4の9.7%）

#### D. SEO / コンテンツマーケティング
- **効果**: 中（長期的には高） / **コスト**: 時間のみ / **時間**: 6〜12ヶ月
- 1〜3ヶ月: 基盤構築期。目に見える成果はほぼない
- 3〜6ヶ月: トラフィック増加が始まる
- 6〜12ヶ月: コンバージョンが発生し始める
- ニッチキーワードは3〜6ヶ月で上位表示の可能性

#### E. マーケットプレイス / ディレクトリ
- **効果**: 中 / **コスト**: 無料〜低額 / **時間**: 即時
- ブラプラ、ゼクシィ等への掲載可否を調査
- Google ビジネスプロフィールの設定（無料）

### CACをゼロに近づける構造的アプローチ

ゼロCACは「獲得チャネルが利用に内蔵されている」こと。

1. ゲストの体験 = プロダクトデモ（50〜100人が毎回体験）
2. SNSシェア = 無料広告（ランキング結果をゲストがシェア）
3. プランナー紹介 = 成果報酬型セールスチーム
4. SEOコンテンツ = 自動集客装置

## 2. ウェディング業界のマーケティング

### 市場概況
- 2025年の婚姻数は約48.5万組の見通し
- 繁忙期: 3〜5月、9〜11月（特に10月・11月が最も人気）
- 閑散期: 6〜8月、12〜2月
- 準備は3〜6ヶ月前から開始 → 繁忙期の3〜5ヶ月前がマーケティング重要時期

### プレ花嫁の情報収集行動

| チャネル | 役割 | 重要度 |
|----------|------|--------|
| **Instagram** | #プレ花嫁 #結婚式準備 で演出・アイデア検索 | 最重要 |
| **ゼクシィ** | 式場検索・比較の定番 | 非常に高い |
| **みんなのウェディング** | 口コミ比較サイト | 高い |
| **ハナユメ** | 式場検索・割引特典 | 高い |
| **Pinterest** | ビジュアルでアイデア収集 | 中〜高 |
| **TikTok/Reels** | 動画コンテンツ。急成長中 | 成長中 |

### 業界の商流

```
カップル → 式場（プランナー）→ 提携業者
                ↑ 紹介手数料（マージン）
```

- 式場プランナー: 提携リストに縛られる。外部サービスは持ち込み料が発生
- フリープランナー: 縛りなし。外部ベンダーを自由に選定可能

**このサービスの構造的優位性**: LINEベースで物理機材の持ち込みが不要。「ゲストが自分のスマホでLINEを使う」行為は持ち込み制限の対象外。

### 競合分析

| サービス | 料金 | 特徴 | 当サービスの優位性 |
|---------|------|------|-----------------|
| フォトシュシュ | 12〜13万円 | 写真投影+フォトコンテスト。テレビ露出あり | 価格1/8。AI採点は唯一無二 |
| 猫の風船 | 無料〜2.9万円 | 業界最安値。SEOブログで集客 | AIゲーミフィケーションで「受動的共有」→「能動的参加」 |
| フォトシャワー | 非公開 | リアルタイム投影 | 同上 |
| WeddingShare | 980〜2,980円 | 写真・動画共有特化 | ランキング+AI評価で圧倒的なエンタメ性 |

**核心的な差別化**: 他サービスは「写真を集める」ツール。当サービスは「ゲスト同士の交流を生み出す」ツール。

### デジタルマーケティングのトレンド（2025-2026年）
1. SEO/コンテンツマーケティングが「非交渉」の最重要施策
2. Instagram Reels/TikTokが急成長。実際の利用シーンを見せることが最も効果的
3. 真正性（Authenticity）の重視。過度にキュレーションされたコンテンツより、リアルな発信
4. 「広く浅く」→「少数のパートナーと深い関係性」がトレンド

## 3. 個人開発プロダクトのグロース

### 「作ったけど誰にも使われない」問題

Indie Hackersの調査によると、ほとんどのプロジェクトは「実行の質」ではなく「ディストリビューション（配布経路）」の欠如で失敗している。ディストリビューションは最終ステップではなく、最初のステップとして扱うべき。

### 最初の10人を獲得した具体事例

| 事例 | 手法 | 所要期間 |
|------|------|---------|
| Proxycurl | コールドメールで10人。パーソナライズした「会話のきっかけ」 | 数週間 |
| Tally | Product Huntで類似ツールにUpvoteしていたユーザーにDM | 数週間 |
| Reddit活用事例 | Redditだけで60人。「売り込み」ではなく「ジャーニーを共有」 | 数ヶ月 |
| AI2sql | 最初の10人に12ヶ月、その後2ヶ月で40人 | 12ヶ月+2ヶ月 |

**共通パターン**: 最初の10顧客は例外なく手動・直接的なアプローチで獲得している。

### コミュニティ活用

#### Product Hunt
- 火〜木の00:01 PST（日本時間16:01）に投稿
- 事前に400人のフォロワーを推奨
- 最初の2時間は全コメントに即レスポンス
- 2〜4週間の事前準備が必要

#### Reddit
- 90/10ルール: 90%は価値提供、10%でプロダクトに触れる
- r/weddingplanning, r/SideProject, r/indiehackers が対象
- 体験談として書く（宣伝臭を出すと即ban）

#### Indie Hackers
- Product Huntより3〜8倍コンバージョンが良いというデータ
- Build in Publicとして進捗を定期共有
- 15〜20分/日の参加から認識を構築

### バイラル係数を上げる設計

**追加すべきバイラル要素:**
1. スコア結果画像に「Powered by Smile Photo Contest」のブランディング
2. 式後にゲスト全員にサービス紹介を自動送信（実装済み）
3. 紹介プログラム（友人紹介で割引）
4. スコア結果のSNSシェアボタン（Instagram Stories用テンプレート）

### 無料→有料への転換

- フリーミアム: 転換率 2〜5%
- トライアル: 転換率 5〜20%
- **推奨**: 最初の5〜10組は完全無料で提供し、レビュー・素材提供を確約してもらう。実績5件後に有料化

## 4. 統合: 戦略的な転換ポイント

3領域の調査を統合して見えた、重要な転換ポイント。

### 転換1: プランナー経由 → カップル直接

プランナーは保守的で新サービスの導入に慎重。DMの反応率も低い。一方、カップルは自分の結婚式を特別にしたいモチベーションが強い。Instagram #プレ花嫁 で直接カップルにリーチする方が効率的。

### 転換2:「機能の説明」→「体験のストーリー」

現在のアウトリーチは機能説明が中心。「自分の結婚式で使ったら、普段写真を撮らない叔父がスマホ片手に走り回ってた」のような感情に訴えるストーリーに変える。

### 転換3: 各イベントが次のマーケティング素材

1回の結婚式 = 50〜100人の体験者 + 写真/動画素材 + 口コミの種。フォトブース業界の成功パターンは「各イベントが次のマーケティング素材」。

### 転換4: Product Hunt / 英語圏への展開

結婚式は世界共通。Product Hunt、Reddit、Indie Hackersでの英語圏ローンチで市場を10倍に広げられる可能性。競合のフォトシュシュ等は日本限定。
